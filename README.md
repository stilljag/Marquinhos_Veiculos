## Instalação

Linux:Dentro do diretório marquinhos_veiculos/backend e frontend

```sh
yarn install 
```

## Configuração para Desenvolvimento

Executar o comando abaixo para rodar o backend.

```sh
yarn dev
```
Executar o comando abaixo para rodar o frontend.

```sh
yarn start
```
